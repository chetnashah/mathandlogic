Set and logic symbols in latex:

Mapping from A to B is $f: A \to B$

If f is injective, it is $f: A \rightarrowtail B$

If f is surjective, it is $f: A \twoheadrightarrow B$

if f is a bijection it is $f: A \leftrightarrow B$
